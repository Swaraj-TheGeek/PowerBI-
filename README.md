# 📊 Power BI Projects

This repository contains my Power BI dashboard projects created for data analysis, visualization, and business intelligence.

## 📁 Projects Included

### 1. HR Analytics Dashboard
**File:** `HR ANALYTICS POWER BI.pbix`

**Dataset:** `HR_Analytics_data.xlsx`

#### Overview
An interactive HR dashboard designed to analyze employee attrition and workforce performance.

#### Features
- Employee Count
- Attrition Analysis
- Attrition Rate
- Age Group Analysis
- Department-wise Attrition
- Job Satisfaction
- Education Field Analysis
- Gender-wise Attrition
- Interactive Filters

#### Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Excel
